# CRM Schema

Each account should contain both company information and AI-generated insights.

| Field | Description |
|-------|-------------|
| Company Name | Practice name |
| Website | Company website |
| Number of Locations | Number of clinics |
| Decision Maker | Practice Owner / Manager |
| Email | Verified email |
| LinkedIn | Profile URL |
| Current Software | If identified |
| Growth Signals | Hiring, expansion, new services |
| Pain Hypothesis | AI-generated assumptions based on public information |
| Lead Score | 0-100 |
| Priority | A / B / C |
| Outreach Angle | Suggested conversation starter |
| AI Summary | One-paragraph account briefing |
| Last Updated | Research date |
| Status | New, Contacted, Discovery, Demo, Closed |
